# Contributing to AgroCarbon AI

Thank you for contributing! Please follow this workflow.

## 🚀 How to Contribute

### 1. Fork the Repository
- Click the **Fork** button at the top right of this repository
- This creates your own copy at: https://github.com/YOUR_USERNAME/AgroCarbonAI

### 2. Clone Your Fork
\\\ash
git clone https://github.com/YOUR_USERNAME/AgroCarbonAI.git
cd AgroCarbonAI
\\\

### 3. Add Upstream Remote
\\\ash
git remote add upstream https://github.com/atul8660/AgroCarbonAI.git
\\\

### 4. Create a Feature Branch
\\\ash
git checkout -b feature/your-feature-name
\\\

### 5. Make Changes
- Write your code
- Test locally
- Commit with clear messages

### 6. Push to Your Fork
\\\ash
git push origin feature/your-feature-name
\\\

### 7. Create a Pull Request
- Go to your fork on GitHub
- Click **Pull Request** → **New Pull Request**
- Base: tul8660/AgroCarbonAI (main branch)
- Head: YOUR_USERNAME/AgroCarbonAI (your feature branch)
- Describe your changes clearly

### 8. Wait for Review
- The maintainer (@atul8660) will review your PR
- Make requested changes if needed
- Once approved, it will be merged

## 📋 Guidelines
- Never push directly to main branch
- One feature per pull request
- Test your code before submitting
- Keep PRs focused and small

## 🔄 Keeping Your Fork Updated
\\\ash
git checkout main
git pull upstream main
git push origin main
\\\
