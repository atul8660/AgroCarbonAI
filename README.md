# AgroCarbon AI - Carbon Biomass Estimation Platform

AI-powered platform for estimating plant biomass from images.

## Features
- User registration with location tracking
- Image upload and storage
- Biomass estimation (simulation)
- MongoDB Atlas cloud database

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript, Bootstrap
- **Backend**: Node.js, Express
- **Database**: MongoDB Atlas
- **Deployment**: Render (backend), Vercel (frontend)

## Future Features
- Real AI model for biomass estimation
- Carbon credit calculation
- Satellite verification
- Admin dashboard

## Live Demo
Frontend: https://agrocarbon.vercel.app
Backend API: https://agrocarbon-api.onrender.com
